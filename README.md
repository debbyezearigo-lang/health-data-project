# Global Health Data Visualization Project

This project analyzes and visualizes global health indicators to explore relationships between healthcare outcomes, economic factors, and country development levels. The dashboard includes an interactive choropleth map, scatterplot, and time-series visualization to examine trends in life expectancy, mortality rates, and health expenditure across countries.

## Technologies Used
- Python
- D3.js
- JavaScript
- HTML/CSS
- Pandas
- GeoJSON

## Features
- Interactive global health visualizations
- Data filtering and tooltips
- Comparative analysis of health and economic indicators
